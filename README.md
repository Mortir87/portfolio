Este repositorio contiene el código fuente de mi portfolio personal, construido como un proyecto propio y mantenido de forma continua.

---

## Sobre el proyecto

Más que una colección de trabajos, este sitio funciona como una carta de presentación interactiva donde comparto proyectos, experiencia y mi forma de entender el desarrollo web.

La prioridad durante el desarrollo ha sido mantener una experiencia rápida, accesible y visualmente cuidada, evitando dependencias innecesarias y apostando por una arquitectura sencilla.

---

## Desarrollo local

```bash
npm install
npm run dev
```

La aplicación estará disponible en:

```text
http://localhost:4321
```

---

## Variables de entorno

Crear un archivo `.env` en la raíz del proyecto:

```env
UPSTASH_REDIS_REST_URL=
UPSTASH_REDIS_REST_TOKEN=
```

---

## Despliegue

El proyecto está preparado para desplegarse en Vercel.

Las variables de entorno deben configurarse también en el panel del proyecto para habilitar el sistema de likes.
